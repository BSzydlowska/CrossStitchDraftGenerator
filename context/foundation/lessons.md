## Never push training materials to GitHub

- **Context**: Committing to repo
- **Problem**: Training materials (10x-cli skills, prompts, internal documentation) published to GitHub expose proprietary content publicly.
- **Rule**: Always add training materials to `.gitignore`, never push them to the GitHub repo.
- **Applies to**: all
